Effect of Wheelchair Use

Nate Lant and Austin Hopkins

4/14/2020

This section compares different models in predicting the likelihood of employment for users with wheelchairs. The response variable is the worker variable which is employment status, 0 for unemployed and 1 for employed.

At the most basic level, the binomial regression is the same as the odds ratio. The logit is \(z\) in \[ z = \beta_0 + \beta_1*x_1 \]

The odds ratio can be calculated using \(e^\beta\). This is not to be confused with the odds, defined as \(e^z\). With odds, we can calculate the probability of getting employed as

\[ P = \frac{e^z}{1+e^z}\]

Reduced Model

The first model will regress worker status on wheelchair usage. This model will calculate the odds of employment if you have a wheelchair compared to not having a wheelchair (odds ratio).

glm(worker ~ wheelchair, data = mydata1, family = binomial(link='logit')) %>%
  summary()

This shows for example the odds ratio of a wheelchair user who attained education level 5 (graduate or professional degree) is \(e^(-3.03+1.03+1.175)\) = 0.438.

This shows that education (with the exception of graduate school) does not affect employment for wheelchair users any differently than for non wheelchair users.
